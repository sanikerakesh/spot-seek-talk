# 🌍 Tour Guide AI

An AI-powered Tourist Guide Web App that helps users discover the best nearby places using location, maps, and intelligent recommendations.

---

## 🚀 Features

* 📍 **Live Location Detection**

  * Automatically detects user location using GPS

* 🗺️ **Interactive Map**

  * Displays user location and nearby places using Google Maps / Leaflet

* 🌐 **Nearby Places Discovery**

  * Finds places within **50–80 km radius**
  * Categories:

    * 🏛️ Temples
    * 🌳 Parks
    * 🍽️ Restaurants
    * 🎯 Hangout spots
    * 🏞️ Tourist attractions
    * 💎 Hidden gems

* 🤖 **AI Recommendations (Gemini)**

  * Ask queries like:

    * “Best temples near me”
    * “Places to visit with friends”
  * Returns top recommendations with reasons

* 🎤 **Voice Input**

  * Speak instead of typing queries

* 🔊 **Text-to-Speech**

  * AI responses are spoken aloud

* 📍 **Map Integration with Markers**

  * Shows recommended places on map

* 🔗 **Google Maps Navigation**

  * Each place includes:

    * Name
    * Rating
    * Address
    * Distance
    * 👉 “Open in Maps” link
